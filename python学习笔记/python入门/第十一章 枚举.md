### 第十一章 Python的高级语法

#### 一、枚举

1.通过枚举的名称可以获取到枚举的类型

| 枚举类型  | 枚举的名称   | 枚举的值       |
| --------- | ------------ | -------------- |
| VIP.GREEN | VIP['GREEN'] | VIP.GREEN.name |

2.枚举中标签不允许同名，报错。

  值相同不会报错，但是相同值的只保留第一个标签，后面的出现值相同的标签会被作为第一个标签的别名，并且在for循环遍历所有标签时会被省略。
