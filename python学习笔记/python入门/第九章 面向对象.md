**六、类的成员变量的操作**

1、不提倡直接由对象来改变成员变量的值，而应该通过实例方法来改变。保证类的安全性。不要在对象的外部直接改变成员变量的值。

2、成员的可见性(即变量和方法的可见性)

public 公开的

private 私有的 变量名签名加双下划线 __
